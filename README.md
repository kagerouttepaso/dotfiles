dotfiles
========

How to Use
----------

**やること**  

```bash
git clone https://github.com/kagerouttepaso/dotfiles
cd dotfiles
./install.sh
```

**注意**  
結構依存しているパッケージがある。
多分この辺だったはず。。。

- zsh
- tmux
- ctags
- pandoc
- vim

ubuntu12.04以上ならこっちを`git clone`してchefを走らせたほうが賢明です
