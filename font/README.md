fonts for Cygwin
===============

How to Use
----------
1. Install this files  
   ./Inconsolata-Regular.ttf  
   ./Inconsolata-Bold.ttf  
   ./Hack-*.ttf
2. Run `./SetInconsolataFont.reg`
